# Marcos Winicyus - Portfolio

This repository contains the source code for my personal portfolio website.

## 🚀 About the Project

This portfolio was built to showcase my journey as a **Senior Analytics Engineer** and **AI Engineer**, highlighting my experience with scalable data platforms, Agentic Workflows, and architecture modernization.

- **Main Technology:** [Astro](https://astro.build/)
- **Styling:** Tailwind CSS (via Astro integration)

## 🛠 How to run locally

If you want to run the project locally on your machine:

1. Install the dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

The site will be available at `http://localhost:4321`.
